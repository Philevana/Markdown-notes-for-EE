# Markdown Notes for EE

面向机器人/自动驾驶方向的系统化学习笔记仓库，采用分篇分章组织，覆盖数学基础、建模、估计、控制、规划、工程化与项目实战。

## 仓库结构

- `01-第一篇-数学优化与数值计算基础/`
- `02-第二篇-机器人与车辆建模基础/`
- `03-第三篇-状态估计与传感器融合/`
- `04-第四篇-控制理论与实现/`
- `05-第五篇-规划决策与导航/`
- `06-第六篇-软件工程系统集成与实时性/`
- `07-第七篇-仿真验证与安全/`
- `08-第八篇-学习控制与具身智能拓展/`
- `09-第九篇-项目实战与求职映射/`
- `10-附录/`

## 顶层文件说明

- `学习大纲.md`：全书结构与章节规划
- `学习笔记模板.md`：统一笔记模板
- `主Prompt-单章生成.md`：单章内容生成主提示词

## 使用方式

1. 先阅读 `学习大纲.md`，明确整体学习路径。
2. 按章节顺序进入各篇目录学习与补充。
3. 使用 `学习笔记模板.md` 保持笔记风格一致。
4. 需要批量生成时参考 `主Prompt-单章生成.md`。

## 版本管理

当前仓库已启用 Git，默认分支为 `main`，并绑定远程 `origin`。

常用命令：

```bash
git status
git add -A
git commit -m "docs: update notes"
git push
```
